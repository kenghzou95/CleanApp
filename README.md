# CleanApp
This repo Contains create react app with mui and echarts
The purpose of this repo is to help people who wanted to start a new project at the minimum effort.

Includes:
MUI v5
echarts v3.0.2
MUIX
moments

It can helps :
- Create UI
- Create Table
- Create Graph
